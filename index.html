📋 Expected Format
Your Excel file should have these columns (any order):

✓ PO No. or PO No
✓ PO Date
✓ Item Desc
✓ Qty
✓ Price
✓ NBD (due date)
✓ Customer or Col 2
✓ Status (optional)
💡 Your existing Kanban Excel file already matches this format — just upload it directly. The importer auto-detects the customer from column 2 and the status from the Delivered/Completed columns.
